# Switch networks

This guide will walk you through the process of changing networks.

1. Open Station and connect your wallet.

2. Click the settings icon in the upper right-hand corner. In Station desktop the settings icon is a globe, in the browser extensions the icon is a gear.

3. Navigate to the **Network** tab and select the network you would like to connect to.
